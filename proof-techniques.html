1. Proof Terminology

  • Theorem: A statement that can be shown to be true.
  • Proof: A valid argument that establishes the truth of a theorem.
  • Axioms: Statements we assume to be logically true (no proof needed).
  • Lemma: A less important theorem that is helpful in the proof of other, larger results.
  • Corollary: A theorem that can be established directly from a theorem that has just been proved.
  • Conjecture: A statement that is being proposed to be a true statement (but hasn't been proven yet).

2. Standard Forms of Numbers

Before proving mathematical statements, you must know how to translate English number properties into algebraic formulas.

Type of Number Standard Algebraic Form
Even number 2k, where k is an integer
Odd number 2k + 1 (or 2k - 1), where k is an integer
Multiple of n kn, where n is an integer
Division by k gives remainder r kn + r, where n is an integer
Perfect square k2, where k is an integer
Rational number p / q, where p, q are integers and q ≠ 0
Irrational number NO STANDARD FORM (Cannot be written as p/q)

3. Vacuous and Trivial Proofs

Recall the truth table for a conditional statement p → q. It is true when p is false, and it is true when q is true. We can use these facts as shortcuts!

Vacuous Proof

If we prove a conditional statement by disproving its hypothesis (showing p is False), the implication is automatically True.

Example: Prove P(0) is true, where P(n) ≡ (n > 1) → (n2 > n).

Proof: For n = 0, the hypothesis (0 > 1) is False. Because a False premise always implies a True conclusion, the statement is vacuously true.

Trivial Proof

If we prove a conditional statement by proving its conclusion (showing q is True), the implication is automatically True.

Example: Prove P(0) is true, where P(n) ≡ (a ≥ b) → (an ≥ bn).

Proof: For n = 0, the conclusion becomes (a0 ≥ b0), which is (1 ≥ 1). This is True. Because the conclusion is True, the statement is trivially true.

4. Direct Proof

A direct proof of a conditional statement p → q assumes that p is true, and uses axioms, definitions, and theorems to show that q must also be true.

Theorem: If n is an even integer, then n2 is even.

Proof:
  1. Assume that n is an even integer. (Assume p is true).
  2. By definition, this implies there is some integer k such that n = 2k.
  3. We want to find n2. So, we square both sides:
    n2 = (2k)2 = 4k2
  4. We need to show this result is even (a multiple of 2). Let's factor out a 2:
    n2 = 2(2k2)
  5. Since k is an integer, 2k2 is also an integer.
  6. Since n2 can be written as 2 times an integer, it follows by definition that n2 is even. [Proved]

5. Proof by Contraposition (Indirect Proof)

Sometimes, p → q is hard to prove directly. Because a statement is logically equivalent to its contrapositive, we can instead prove ¬q → ¬p.

Steps: Assume q is False, and prove that p must be False.

Theorem: If n is an integer and n2 is odd, then n is odd.

Proof:

Instead of proving p → q (If n2 is odd → n is odd), we will prove the contrapositive ¬q → ¬p (If n is even → n2 is even).

  1. Assume the conclusion is false: n is an even integer.
  2. By definition, n = 2k for some integer k.
  3. Then, n2 = (2k)2 = 4k2 = 2(2k2).
  4. Since n2 is 2 times an integer, n2 is even.
  5. Since the contrapositive is true, the original statement is also true. [Proved]

6. Proof by Contradiction

To prove a statement p is true, we assume the opposite (¬p is true). We then show that this assumption logically leads to a mathematical impossibility or contradiction. Therefore, our assumption must be false, meaning the original statement must be true.

For a conditional statement "If p, then q", the negation is p ∧ ¬q. So we assume p is true AND q is false, and look for a contradiction.

Theorem: Prove that √2 is irrational.

Proof:
  1. Assume the opposite is true: √2 is a rational number.
  2. By definition, √2 can be expressed as a fraction p/q, where p and q are integers, q ≠ 0, and they have no common factors other than 1 (the fraction is fully simplified).
  3. So, √2 = p/q
  4. Square both sides: 2 = p2 / q2
  5. Multiply by q2: p2 = 2q2
  6. This means p2 is an even number. If a square is even, the root is even. Therefore, p must be even. Let p = 2k.
  7. Substitute p=2k back into the equation: (2k)2 = 2q2 → 4k2 = 2q2
  8. Divide by 2: 2k2 = q2
  9. This means q2 is even. Therefore, q must be even.
  10. The Contradiction: We just showed that both p and q are even. This means they share a common factor of 2. But in step 2, we stated p and q have NO common factors.
  11. Because our assumption led to a contradiction, the assumption must be false. Therefore, √2 is irrational. [Proved]
