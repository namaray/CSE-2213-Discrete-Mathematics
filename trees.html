1. What is a Tree?

A tree is a connected undirected graph with no simple circuits. It cannot contain multiple edges or loops.

A forest is a collection of one or more disjoint trees where each tree is independent of the others, and there are no edges connecting the trees.

Rooted Trees & Terminology

  • Rooted Tree: A tree in which one vertex is designated as the root, and every edge is directed away from the root.
  • Root: The starting point, has no parent.
  • Leaf: A node with no children (the end of the branch).
  • Internal Vertices: Vertices that have children. The root is an internal vertex unless it's the only node in the tree.
  • Ancestors/Descendants: Nodes above/below a given node in the path to the root/leaves.
  • Level: How far a node is from the root (number of edges). Root is at level 0.
  • Height/Depth: The number of edges on the longest path from the root to a leaf (the maximum level).

2. m-ary and Binary Trees

An m-ary tree is a rooted tree where every internal vertex has no more than m children.

It is called a full m-ary tree if every internal vertex has EXACTLY m children.

Binary Search Tree (BST)

An m-ary tree where m = 2 is a binary tree. A Binary Search Tree (BST) is a special binary tree that follows a specific ordering rule:

  • The left subtree contains values less than the node's value.
  • The right subtree contains values greater than the node's value.

Note: For strings, use dictionary order (e.g., center < centre < dictator).

Balanced Tree:
A tree is balanced if all of its leaves are at level h or h - 1 (where h is the height of the tree).

3. Important Tree Theorems

Theorem 2: A tree with n vertices has n - 1 edges.
Theorem 3: A full m-ary tree with i internal vertices contains a total of n = mi + 1 vertices.
Theorem 4: A full m-ary tree with i internal vertices contains a total of l = (m - 1)i + 1 leaves.
Theorem 5: An m-ary tree of height h has at most mh leaves. (Lmax = mh).
Example Problem: Calculate the number of vertices in a full 5-ary tree with 35 internal vertices. Find the number of leaves.
Solution: m = 5, i = 35
Vertices n = mi + 1 = 5(35) + 1 = 176
Leaves l = (m-1)i + 1 = (4)(35) + 1 = 141

4. Tree Traversals & Expressions

Tree traversal refers to visiting all nodes in a specific order. The three standard orders are:

  • Pre-order: Root, Left, Right
  • In-order: Left, Root, Right
  • Post-order: Left, Right, Root
Inorder in BST:
An inorder traversal of a Binary Search Tree ALWAYS produces a sorted sequence of the data. This acts as a correctness test!

Ordered Rooted Tree of Expressions

Also known as a Binary Expression Tree. It represents mathematical operations where internal nodes are operators (+, -, *, /) and leaves are operands (numbers, variables).

Traversing an expression tree yields different notations:

  • In-order: Gives Infix notation (e.g., x + y * 2).
  • Pre-order: Gives Prefix notation (e.g., + x * y 2).
  • Post-order: Gives Postfix notation (e.g., x y 2 * +).
