1. Introduction to Trees

A tree is a connected undirected graph with no simple circuits.

  • Because a tree cannot have a simple circuit, a tree cannot contain multiple edges or loops. Therefore, any tree must be a simple graph.
Theorem:
An undirected graph is a tree if and only if there is a unique simple path between any two of its vertices.

Forest: A graph that has no simple circuits but is not necessarily connected. Each of its connected components is a tree.

2. Rooted Trees

A rooted tree is a tree in which one vertex has been designated as the root and every edge is directed away from the root.

Terminology

  • Parent: The vertex u such that there is a directed edge from u to v.
  • Child: If u is the parent of v, then v is the child of u.
  • Siblings: Vertices with the same parent.
  • Leaf: A vertex with no children (degree = 1).
  • Internal Vertex: A vertex that has at least one child.
  • Ancestors & Descendants: Vertices on the path from the root to v / vertices below v.

3. Properties of Trees

An m-ary tree is a rooted tree where every internal vertex has no more than m children. It is a full m-ary tree if every internal vertex has exactly m children. An m-ary tree with m = 2 is called a binary tree.

Important Theorems:
  1. A tree with n vertices has exactly n - 1 edges.
  2. A full m-ary tree with i internal vertices contains n = m*i + 1 vertices.
  3. There are at most mh leaves in an m-ary tree of height h.

4. Tree Traversal

Tree traversal algorithms provide a process for systematically visiting every vertex of an ordered rooted tree.

Universal Address System & Traversals

  • Preorder Traversal: Visit the root, traverse the left subtree, traverse the right subtree. (Root, Left, Right)
  • Inorder Traversal: Traverse the left subtree, visit the root, traverse the right subtree. (Left, Root, Right)
  • Postorder Traversal: Traverse the left subtree, traverse the right subtree, visit the root. (Left, Right, Root)
Applications:
Traversals are used heavily in computer science for parsing arithmetic expressions (Prefix, Infix, Postfix notations).
