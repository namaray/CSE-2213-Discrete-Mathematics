1. Basics of Counting

The Product Rule

If a procedure can be broken down into a sequence of two tasks, where the first task can be done in n1 ways and the second task can be done in n2 ways, then there are n1 × n2 ways to do the procedure.

The Sum Rule

If a task can be done either in one of n1 ways or in one of n2 ways (where the sets of ways are disjoint), then there are n1 + n2 ways to do the task.

Principle of Inclusion-Exclusion (Subtraction Rule):
If the sets of ways to do tasks are NOT disjoint, you must subtract the overlap.
|A ∪ B| = |A| + |B| - |A ∩ B|

2. The Pigeonhole Principle

If k is a positive integer and k + 1 or more objects are placed into k boxes, then there is at least one box containing two or more of the objects.

Generalized Pigeonhole Principle:
If N objects are placed into k boxes, then there is at least one box containing at least ⌈N/k⌉ objects.

3. Permutations (Order Matters)

A permutation of a set of distinct objects is an ordered arrangement of these objects.

Formula for r-Permutations

The number of r-permutations of a set with n distinct elements is denoted by P(n, r).

P(n, r) = n! / (n - r)!

Example: How many ways can 3 students be selected for 1st, 2nd, and 3rd prize out of 10 students?
P(10, 3) = 10! / (10 - 3)! = 10 × 9 × 8 = 720

4. Combinations (Order Doesn't Matter)

An r-combination of elements of a set is an unordered selection of r elements from the set.

Formula for r-Combinations

The number of r-combinations of a set with n distinct elements is denoted by C(n, r) or (n choose r).

C(n, r) = n! / (r! × (n - r)!)

Example: How many ways can a committee of 3 students be chosen from a group of 10?
C(10, 3) = 10! / (3! × 7!) = (10 × 9 × 8) / (3 × 2 × 1) = 120

Note: C(n, r) = C(n, n-r). Choosing r elements to include is the same as choosing n-r elements to leave out.
