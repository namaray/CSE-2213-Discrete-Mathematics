1. Introduction to Graphs

A graph G = (V, E) consists of V, a nonempty set of vertices (or nodes) and E, a set of edges. Each edge has either one or two vertices associated with it, called its endpoints.

Types of Graphs

  • Simple Graph: Undirected edges, no multiple edges between the same two vertices, no loops (edges from a vertex to itself).
  • Multigraph: Undirected edges, multiple edges allowed between the same vertices.
  • Directed Graph (Digraph): Edges have direction (indicated by arrows).
Handshaking Theorem:
For any undirected graph with e edges, the sum of the degrees of the vertices is 2e.
∑ deg(v) = 2e

2. Special Graph Structures

Complete Graphs (Kn)

A simple graph that contains exactly one edge between each pair of distinct vertices. It has n(n-1)/2 edges.

Bipartite Graphs

A graph whose vertex set can be partitioned into two disjoint sets V1 and V2 such that every edge connects a vertex in V1 to one in V2. (No edges within the same set).

Graph Isomorphism:
Two graphs G1 and G2 are isomorphic if there is a one-to-one and onto function from the vertices of G1 to the vertices of G2 that preserves adjacency. (They have the same structure but different labels).

3. Paths and Connectivity

A path is a sequence of edges that begins at a vertex and travels from vertex to vertex along edges. A path is a circuit (or cycle) if it begins and ends at the same vertex.

  • Connected Graph: There is a path between every pair of distinct vertices.
  • Strongly Connected (Directed): There is a path from a to b and from b to a for all a,b.

4. Eulerian and Hamiltonian Paths

Euler Paths & Circuits (Edge Focus)

An Euler circuit in a graph G is a simple circuit containing every edge of G exactly once.

  • Theorem: A connected multigraph has an Euler circuit if and only if every vertex has an even degree.

An Euler path in a graph G is a simple path containing every edge exactly once.

  • Theorem: A connected multigraph has an Euler path but not an Euler circuit if and only if it has exactly two vertices of odd degree.

Hamilton Paths & Circuits (Vertex Focus)

A Hamilton circuit is a circuit that passes through every vertex exactly once (except for the start/end vertex).

A Hamilton path is a path that passes through every vertex exactly once.

(Unlike Euler circuits, there are no simple necessary and sufficient criteria to determine if Hamilton circuits exist).
