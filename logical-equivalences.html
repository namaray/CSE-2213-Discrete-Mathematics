Logical Equivalence

1. What is Logical Equivalence?

Two logical statements P and Q are said to be logically equivalent if, and only if, they have identical truth values in every possible interpretation.

In symbolic terms, we write this as: P ≡ Q

This means that the biconditional P ↔ Q is always true, regardless of the truth values of the individual components.

Tautology vs. Contradiction

  • Tautology: A compound proposition that is ALWAYS true.
    Example: P ∨ ¬P (The light is either on, or it is not on).
  • Contradiction: A compound proposition that is ALWAYS false.
    Example: P ∧ ¬P (The light is on and not on at the same time).

2. Proving Equivalence using Truth Tables

You can prove P ≡ Q by showing that the column for P exactly matches the column for Q in a truth table.

Example: Prove p → q ≡ ¬p ∨ q

p q ¬p p → q ¬p ∨ q
TTFTT
TFFFF
FTTTT
FFTTT

Because the last two columns match perfectly, the equivalence is proven!

3. Logical Equivalence Laws (The "Cheat Sheet")

Instead of drawing giant truth tables, we can use these algebraic laws to simplify expressions.

Law Name Representation (AND / OR)
Identity p ∧ T ≡ p
p ∨ F ≡ p
Domination p ∨ T ≡ T
p ∧ F ≡ F
Idempotent p ∧ p ≡ p
p ∨ p ≡ p
Negation p ∧ ¬p ≡ F
p ∨ ¬p ≡ T
Double Negation ¬(¬p) ≡ p
Commutative p ∧ q ≡ q ∧ p
p ∨ q ≡ q ∨ p
Associative (p ∧ q) ∧ r ≡ p ∧ (q ∧ r)
(p ∨ q) ∨ r ≡ p ∨ (q ∨ r)
Distributive p ∧ (q ∨ r) ≡ (p ∧ q) ∨ (p ∧ r)
p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r)
De Morgan’s Laws ¬(p ∧ q) ≡ ¬p ∨ ¬q
¬(p ∨ q) ≡ ¬p ∧ ¬q
Absorption p ∧ (p ∨ q) ≡ p
p ∨ (p ∧ q) ≡ p

Special Laws (Implications & Biconditionals)

Conditional Law p → q ≡ ¬p ∨ q
Contrapositive Law p → q ≡ ¬q → ¬p
Biconditional Law p ↔ q ≡ (p → q) ∧ (q → p)

4. Step-by-Step Propositional Equivalences

Using the laws above, we can prove complex equivalences without truth tables.

Example 1: Prove that ¬(p → q) ≡ p ∧ ¬q

¬(p → q)
≡ ¬(¬p ∨ q) [Conditional Law]
≡ ¬(¬p) ∧ ¬q [De Morgan's Law]
≡ p ∧ ¬q [Double Negation]

Example 2: Prove that (p ∧ q) → (p ∨ q) is a Tautology

(p ∧ q) → (p ∨ q)
≡ ¬(p ∧ q) ∨ (p ∨ q) [Conditional Law]
≡ (¬p ∨ ¬q) ∨ (p ∨ q) [De Morgan's Law]
≡ (¬p ∨ p) ∨ (¬q ∨ q) [Commutative & Associative Laws]
≡ T ∨ T [Negation Law]
≡ T [Domination Law]

Example 3: Prove that (p → q) ∧ (p → r) ≡ p → (q ∧ r)

(p → q) ∧ (p → r)
≡ (¬p ∨ q) ∧ (¬p ∨ r) [Conditional Law]
≡ ¬p ∨ (q ∧ r) [Distributive Law (in reverse)]
≡ p → (q ∧ r) [Conditional Law]
