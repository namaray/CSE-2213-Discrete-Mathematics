1. Introduction to Sets

A set is a collection of unordered objects. Objects in a set are called members or elements.

  • a ∈ V means "a is a member of V".
  • x ∉ V means "x is not a member of V".

Important Mathematical Sets

Symbol Name Elements
ℕNatural Numbers{0, 1, 2, 3, ...}
ℤIntegers{..., -2, -1, 0, 1, 2, ...}
ℤ+Positive Integers{1, 2, 3, ...}
ℚRational Numbers{p/q | p∈ℤ, q∈ℤ, q≠0}
ℝReal NumbersAll numbers on the number line

Ways to represent a set

  • Roster Method: Listing all elements explicitly.
    Example: S = {1, 2, 3, 4, 5}
  • Set Builder Notation: Stating the properties that elements must satisfy.
    Example: S = {x ∈ ℤ+ | x ≤ 10}

2. Subsets & The Empty Set

Subsets (⊆)

A subset is a set where all of its elements are present within another set.

If A={1,2,3} and B={1,2,3,4,5}, then A ⊆ B.

Proper Subsets (⊂)

If A is a subset of B, but A and B are not the exact same set, A is a proper subset.

Condition: A ⊆ B AND A ≠ B.

The Empty Set (∅)

An empty set (or null set) contains no elements. It is denoted by ∅ or { }.

  • The empty set is a subset of every set. ∅ ⊆ A
  • Careful: {∅} is NOT an empty set. It is a set containing one element (the empty set).

3. Cardinality & Power Sets

Cardinality

For a finite set S, the cardinality is the number of distinct elements it contains, denoted by |S|.

  • If V = {a, e, i, o, u}, then |V| = 5
  • The cardinality of the empty set is 0: |∅| = 0

Power Set

The power set of S, denoted P(S), is the set of all possible subsets of S.

Example: What is the power set of {1, 2}?
P({1, 2}) = { ∅, {1}, {2}, {1, 2} }
Important Formula: If a set S has n elements, its Power Set has 2n elements.
|P(S)| = 2|S|

Example: What is the cardinality of P(∅)?
Since |∅| = 0, the cardinality of P(∅) = 20 = 1.

4. Cartesian Product

The Cartesian product of A and B, denoted A × B, is the set of all possible ordered tuples (x, y) where x is from A and y is from B.

A × B = { (x,y) | x ∈ A ∧ y ∈ B }

Example: Let A = {1, 2, 3} and B = {a, b}
A × B = {(1,a), (1,b), (2,a), (2,b), (3,a), (3,b)}
  • Order matters! A × B ≠ B × A
  • Cardinality Formula: |A × B| = |A| * |B|

5. Set Operations

1. Union (A ∪ B)

Combines all elements from both sets (no duplicates).

A ∪ B = {x | x ∈ A ∨ x ∈ B}

Ex: {1,2,3} ∪ {1,3,5} = {1,2,3,5}

2. Intersection (A ∩ B)

Only the elements that are common to both sets.

A ∩ B = {x | x ∈ A ∧ x ∈ B}

Ex: {1,2,3} ∩ {1,3,5} = {1,3}

Note: Two sets are disjoint if A ∩ B = ∅.

3. Difference (A − B)

Elements in A but NOT in B (Relative complement).

A − B = {x | x ∈ A ∧ x ∉ B}

Ex: {1,2,3} − {1,3,5} = {2}

4. Complement (A' or Ā)

Elements in the Universal Set (U) that are NOT in A.

A' = U − A

Principle of Inclusion and Exclusion:
To find the cardinality of a union set, use this formula:
|A ∪ B| = |A| + |B| − |A ∩ B|

6. Truth Sets of Predicates

The truth set of a predicate is the set of all elements in the domain that make the predicate true.

T = {x ∈ D | P(x) is true}

  • What is the truth set of P(x) ≡ |x| = 1?
    Answer: {1, -1}
  • What is the truth set of Q(x) ≡ |x| = x?
    Answer: [0, ∞) (All non-negative real numbers)
  • What is the truth set of R(x) ≡ x + 1 > x?
    Answer: ℝ (The set of all real numbers)
