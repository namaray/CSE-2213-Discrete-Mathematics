1. Introduction to Functions

Let A and B be non-empty sets. A function f from set A to set B is the mapping of every element of A to a single unique element of B.

  • Set Notation: f : A → B
  • Value Notation: f(a) = b (if the function f maps the value a ∈ A to a value b ∈ B)
CAUTION: If f(x) returns multiple values for the same value of x, it is NOT a function. Every member of the domain must map to exactly one unique element in the codomain.

Domain, Codomain, and Range

For a function f : M → T:

  • Domain (M): The set of all possible input values.
  • Codomain (T): The set of all potential output values.
  • Range / Image (R): The set of only those elements that are actual outputs of f. R ⊆ T.

2. Properties of Functions

One-to-One (Injection)

A function maps distinct elements of its domain to distinct elements of its codomain. No two different inputs are mapped to the same output.

Check: For all x1, x2 ∈ A, if f(x1) = f(x2), then x1 = x2.

Onto (Surjection)

Every element in the codomain has at least one pre-image in the domain. In other words, the Codomain is equal to the Range.

Check: For each y ∈ B, there exists x ∈ A such that f(x) = y.

Bijections

A bijection (or bijective function) is a function that is both one-to-one and onto. It establishes a perfect "one-to-one correspondence" between elements of its domain and elements of its codomain.

Detecting One-to-One and Onto algebraically

Example 1: f(x) = x2 where f : ℤ → ℤ

  • One-to-one? No. f(2) = 4 and f(-2) = 4. Since 2 ≠ -2, different inputs yield the same output.
  • Onto? No. The codomain is ℤ, but f(x) only produces non-negative integers. There is no integer x such that x2 = -1.

Example 2: f(x) = x + 1 where f : ℝ → ℝ

  • One-to-one? Yes. If f(x1) = f(x2), then x1 + 1 = x2 + 1 ⇒ x1 = x2.
  • Onto? Yes. Let y ∈ ℝ be any arbitrary element in the codomain. Then x = y - 1 is a real number, and f(x) = (y - 1) + 1 = y.

3. Operations on Functions

Addition and Multiplication

Functions of the same domain can be added or multiplied:

  • Addition: (f + g)(x) = f(x) + g(x)
  • Multiplication: (fg)(x) = f(x)g(x)

Example: If f(x) = x2 and g(x) = x - x2, then (f + g)(x) = x and (fg)(x) = x3 - x4.

Inverse of a Function

The inverse of a function exists if and only if the function is a bijection (called an invertible function).

If f(a) = b, then the inverse function f-1 is defined as f-1(b) = a.

How to find the inverse:
  1. Replace f(x) with y.
  2. Interchange x and y.
  3. Solve the equation for y.
  4. Replace y with f-1(x).

Example 1: Find the inverse of f(x) = ex.

  • y = ex
  • x = ey (Interchange)
  • ln(x) = ln(ey) ⇒ ln(x) = y
  • f-1(x) = ln(x)

Example 2: Find the inverse of f(x) = (2x + 1)/(x - 1).

  • y = (2x + 1)/(x - 1)
  • x = (2y + 1)/(y - 1) (Interchange)
  • x(y - 1) = 2y + 1 ⇒ xy - x = 2y + 1 ⇒ y(x - 2) = x + 1
  • f-1(x) = (x + 1)/(x - 2)
